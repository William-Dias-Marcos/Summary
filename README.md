## ⚙️ Como rodar o projeto

Necessário ter o Node.js em sua máquina e o Angular CLI.

- git clone do repositorio
- npm install
  
Para iniciar o Server:
- npm run dev
  
Para iniciar a aplicação em angular:
- ng serve
